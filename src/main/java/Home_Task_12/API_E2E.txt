addUser
Метод добавляет нового пользователя и возвращает доступ к личному кабинету пользователя
logIn
Метод осуществляет авторизацию пользователя и возвращает доступ к личному кабинету пользователя
deleteByEmail
Метод удаляет пользователя по электронному адресу из базы данных
getAllDevices
Метод получает списоук всех зарегистрированных устройств(пылесос, моющий пылесос) у пользователя
addNew device
Метод добавляет новое устройство пользователлю и обнавляет список зарегистрированных устройств
deleteDevice
Метод удаляет устройство по ID номеру устройства и обнавляет список зарегистрированных устройств
getAllSchedules
Метод получает список запрограмированных пользователем уборок по расписанию (по времени ии дате)
addSchedule
Метод добавляет новое расписаниие уборки по времени и дате
deleteSchedule
Метод удаляет расписание уборки по ID
